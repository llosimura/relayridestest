# Prerequisites
  * nodejs & npm
  * bower
  * gulp

# Develop
To run the app in a development environment with browsersync just run
    gulp serve
If you want to run with a minified version of scripts and css just run
    gulp serve
# Building

To build this app just run
    gulp build
